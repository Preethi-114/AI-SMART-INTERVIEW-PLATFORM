# AI Smart Interview Platform

AI-based mock interview platform built using MERN stack.

## Technologies Used

- React.js
- Node.js
- Express.js
- MongoDB

## Features

- User Authentication
- Mock Interviews
- AI Responses
- Interactive UI

## Project Setup

1. Clone repository

git clone <repository-link>

2. Install dependencies

Frontend:

cd ai-interview-platform
npm install

Backend:

cd backend
npm install

3. Run Project

Frontend:

npm run dev

Backend:

npm start

4. Open Browser

Frontend:

http://localhost:5173

Backend:

http://localhost:5000
